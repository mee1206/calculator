# 🧮 Basic Calculator

A simple and responsive **Basic Calculator** built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations with a clean user interface and keyboard support.

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📊 Modulus (%)
- 🔢 Decimal number support
- 🗑️ Clear (C) button
- ⌫ Delete last character
- ⌨️ Keyboard support
- 📱 Responsive design
- 🎨 Modern UI using CSS Grid

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 💻 Keyboard Shortcuts

| Key | Function |
|------|----------|
| 0–9 | Enter numbers |
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| % | Modulus |
| . | Decimal |
| Enter | Calculate |
| Backspace | Delete last character |
| Esc | Clear display |
